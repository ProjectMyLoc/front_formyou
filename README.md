## Installation

`git clone git@github.com:ProjectMyLoc/front_formyou.git`

`cd front_formyou`

`npm install`

`echo REACT_APP_API_URL='http://localhost:8080' > .env`

`npm start`

Your website is now running on http://localhost:3000
